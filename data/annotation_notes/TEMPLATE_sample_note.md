# Template Sample Note

- 文件用途：只在复杂样本时使用，用来补充 CSV 放不下的判断依据。
- 什么时候使用：样本复杂、字段不确定、存在争议，或需要复核时使用；不是每篇都必须建。
- 我应该怎么维护：复制本模板并改名为 `SXXX_sample_note.md`；每个部分都可以写 `无`。

## Basic Info

可写 `无`。
- `sample_id`：
- `pdf_path`：
- `annotation_status`：
- `date`：

## Author Block Observation

可写 `无`。
- 这里写作者顺序是否清晰、有没有脚注标记、有没有共同一作说明。

## Email Observation

可写 `无`。
- 这里写邮箱位置、共享邮箱、分组邮箱、团队邮箱或断行邮箱情况。

## Ground Truth Decision

可写 `无`。
- 这里写 authors、first_author、co_first_authors 为什么这样填。

## Pairing Notes

可写 `无`。
- 这里写哪些 pair 明确成立，哪些故意不填。

## Region Notes

可写 `无`。
- 这里写 region 为什么填 `CN`、`OVERSEAS` 或 `UNKNOWN`。

## Open Questions

可写 `无`。
- 这里写仍需复核的问题。
